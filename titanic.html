Estudio de la base de datos titanic

En este proyecto se realizó el estudio de la base de datos Titanic, donde se dividirá el proyecto en tres:

• Primera parte: vamos a trabajar algunos de los aspectos más importantes del proceso de descubrimiento de conocimiento a partir de datos (KDD): almacenamiento y carga de datos, análisis exploratorio de datos, preprocesamiento de datos, validación de modelos.

• Segunda parte: estudiaremos los modelos más utilizados en scikit-learn para conocer los distintos hiperparámetros que los configuran y estudiar los clasificadores resultantes. Además, veremos métodos de selección de modelos orientados a obtener una configuración óptima de hiperparámetros.

• Tercera parte: En esta práctica trabajaremos con modelos basados en redes neuronales utilizando, para ello, la librería tensorflow.
