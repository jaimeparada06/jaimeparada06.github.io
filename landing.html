Landing page con HTML5, CSS3 y JavaScript

En este proyecto realizaremos un sitio web, la cual será una landing page utilizando HTML5, CSS3 y JavaScript.

La landing page será de una posible página donde se organizarán y se jugarán torneos relacionados con el mundo de los videojuegos. Y en la cual, se le informará al usuario de las características de la página web asi como, de preguntas frecuentes. Cabe destacar, que esta página web solo se podrá visualizar correctamente en pantallas de ordenador, ya que, no se realizó un diseño responsive para los dispositivos móviles.
