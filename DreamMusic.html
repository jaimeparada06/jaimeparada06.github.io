Web app DreamMusic

En este proyecto realizaremos una web app, donde podremos comprar, alquilar o restaurar discos de un determinado catálogo. Utilizaremos distintos lenguajes de programación como puede ser:

• C#

• CSHTML

• CSS

• JavaScript

• SQL

Además, utilizamos los servicios que ofrece Azure DevOps y una metología Scrum.
